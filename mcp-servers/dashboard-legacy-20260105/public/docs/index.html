dashboard (Mission Control)

REST Express
â–¼

REST API + WebSocket server for the Mission Control dashboard. Real-time work effort monitoring across repositories.

Endpoints

GET /api/health Server health check
GET /api/repos List all monitored repos
GET /api/repos/:name Get single repo state
POST /api/repos Add new repository {name, path}
DELETE /api/repos/:name Remove repository
PATCH /api/repos/:name/work-efforts/:weId/status Update work effort status
GET /api/browse?path=... Browse directories
POST /api/repos/bulk Add multiple repos

WebSocket Events

Connect to ws://localhost:3847 for real-time updates.

WS init Initial state on connection
WS update Repo state changed
WS repo_change Repo added/removed
WS hot_reload UI file changed (dev)
